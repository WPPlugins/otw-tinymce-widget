* Add a title to the widget 
* All tinyMCE options are available 
* Visual or Text mode
* Custom CSS Class

**Custom styling**

If you need to further style the widget content generated by the OTW TinyMCE Widget here are your options:

* Each OTW TinyMCE Widget has it's unique CSS class that can be used to style all OTW TinyMCE Widgets.
* Create a new class for each instance of a OTW TinyMCE Widget in its interface so you can style it individually.
